# Condomiinus - Sistema de Gestão de Condomínios

Sistema completo de gestão condominial desenvolvido com Vue 3, TypeScript e Vite.

## 🚀 Funcionalidades

- **Gestão de Condomínios**: Cadastro e gerenciamento de condomínios
- **Gestão de Unidades**: Cadastro e gerenciamento de unidades (apartamentos, casas, salas)
- **Gestão de Moradores**: Cadastro e gerenciamento de moradores
- **Gestão de Visitantes**: Cadastro e gerenciamento de visitantes
- **Gestão de Reservas**: Reserva de áreas comuns
- **Gestão de Ocorrências**: Registro e acompanhamento de ocorrências
- **Gestão de Funcionários**: Cadastro e gerenciamento de funcionários
- **Gestão de Fornecedores**: Cadastro e gerenciamento de fornecedores
- **Gestão de Serviços**: Cadastro e gerenciamento de serviços
- **Gestão de Despesas**: Controle de despesas condominiais
- **Gestão de Receitas**: Controle de receitas condominiais
- **Gestão de Assembleias**: Registro e acompanhamento de assembleias
- **Gestão de Vagas**: Controle de vagas de garagem
- **Gestão de Animais**: Registro de animais de estimação
- **Gestão de Visitantes**: Controle de visitantes
- **Gestão de Visitantes**: Controle de visitantes

## 🛠️ Tecnologias Utilizadas

- **Frontend**: Vue 3, TypeScript, Vite
- **Backend**: Node.js, Express, TypeScript
- **Database**: PostgreSQL
- **Authentication**: JWT, bcrypt
- **Testing**: Vitest, Playwright
- **UI**: Tailwind CSS, DaisyUI

## 📋 Pré-requisitos

- Node.js 18 ou superior
- PostgreSQL 14 ou superior
- npm ou yarn

## 🚀 Instalação

1. Clone o repositório:
```bash

```

2. Instale as dependências do frontend:
```bash
cd frontend
npm install
```

3. Instale as dependências do backend:
```bash
cd backend
npm install
```

4. Configure o banco de dados:
```bash
cd backend
npm run db:create
npm run db:migrate
```

5. Configure as variáveis de ambiente:
```bash
cd backend
cp .env.example .env
```

6. Inicie o servidor:
```bash
cd backend
npm run dev
```

7. Inicie o frontend:
```bash
cd frontend
npm run dev
```
